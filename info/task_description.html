In a given word you need to check if one symbol goes after another.

Cases you should expect while you solving this challenge

  • one or another symbols are not in the given word - your function should return false
  • any of symbols can be more than once in a word - use only the first one
    • if two symbols are the same - your function should return false
  • condition is case sencetive. It mease 'a' and 'A' are two different symbols

Input: Three arguments. First is a given string, second is a symbol that shoud go first and the third one is a symbold that should go after.

Output: Bool.
